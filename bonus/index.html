Game exemples

How to move.

To move, you need to use the four arrows key on your keyboard to move to right, left, up, or down. Thats the only key that will make you move.

Restart map.

You failed something? You found out a way to end the map you're actually playing in ? Just press the space keyboard and the game will restart.

Win and lose.

If you block all the 'X', the game will automatically close. If you put every 'X' on an 'O', the game will leave showing you that you won. Congrats !
